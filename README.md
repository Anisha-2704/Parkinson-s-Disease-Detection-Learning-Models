# Parkinson's Disease Detection from EEG Signals using Learning Algorithms

>Objective:

To develop a machine learning-based framework for detecting Parkinson’s disease using EEG signals under a robust subject-wise evaluation strategy.

>Technologies Used:

Python, NumPy, Pandas, Scikit-learn, Matplotlib, EEG signal processing techniques, Machine Learning algorithms (SVM, KNN, Random Forest, XGBoost, LightGBM, CatBoost).

>What We Built:

A complete EEG analysis pipeline involving preprocessing, feature extraction, subject-wise validation, and comparative evaluation of multiple machine learning models for Parkinson’s disease detection.

>Outcome / Results:

Ensemble models, especially Random Forest and CatBoost, demonstrated strong and consistent performance across datasets, while subject-wise validation provided a more reliable evaluation of model generalization.
